# PyGen Product Guidelines

## Brand Messaging & Tone
- **Tone:** Technical and Efficient.
- **Messaging:** Emphasize the speed, automation, and reliability of the platform. PyGen is a professional tool for high-volume, high-quality Islamic video production.

## Visual Identity & UI Design
- **Theme:** Modern and Bold.
- **Aesthetic:** Use strong contrasts and vibrant color palettes to convey power and speed. The interface should feel dynamic and high-performance.
- **Layout:** Prioritize clarity and rapid access to core functions and status information.

## User Interaction & Feedback
- **Feedback Loops:** Utilize real-time progress bars for all processing and upload tasks to provide immediate visual confirmation of activity and speed.
- **Transparency:** Provide detailed activity logs for every job. Users should have granular insight into the technical steps of video generation and YouTube integration.
- **Status Indicators:** Use high-contrast visual cues to represent different job states (e.g., active processing, completed, error).

## Documentation & Support
- **Format:** Centralized Technical Guide.
- **Content:** Maintain a high-density, technical documentation page that covers configuration, API integrations, and troubleshooting in detail. Avoid overly flowery language in favor of direct, actionable technical information.
