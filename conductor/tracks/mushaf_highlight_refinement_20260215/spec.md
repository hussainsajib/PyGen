# Specification: Mushaf Video Highlighting & Integration Refinement

## Overview
This track addresses visual and functional refinements for Mushaf video generation, focusing on fixing the highlighting transparency, improving terminal logging for file exports, and ensuring seamless integration between high-speed engines and social media upload workflows.

## Functional Requirements
1.  **Highlighting Fix:**
    -   Investigate and fix the issue where highlight transparency (intended approx. 30%) is not taking effect in the video output.
    -   Ensure the final effect is a clearly visible but semi-transparent yellow overlay on the active line.
2.  **Terminal Path Logging:**
    -   Modify the video generation process to log the **relative disk path** of the exported video file directly to the terminal upon successful completion.
    -   Applies to standard Mushaf, Juz, and high-speed engines.
3.  **Social Media Integration Audit:**
    -   **Config Connectivity:** Verify that UI toggles for "YouTube Upload" and "Facebook Upload" correctly propagate to the background worker for high-speed engines.
    -   **Metadata Integrity:** Ensure high-speed engines generate the required `.txt` details files in `exported_data/details` to power social media descriptions.
    -   **Error Visibility:** Ensure upload failures are properly captured and reported in the job status.

## Non-Functional Requirements
-   **Visual Consistency:** The fixed highlight should look consistent across all rendering backends (MoviePy, FFmpeg, OpenCV, PyAV).
-   **Logging Clarity:** Terminal logs should be easy to read and identify during high-volume generation.

## Acceptance Criteria
-   [ ] Active lines in Mushaf videos feature a correctly applied semi-transparent yellow highlight.
-   [ ] Successful video generation jobs print the relative file path to the terminal.
-   [ ] High-speed engine jobs successfully trigger automated uploads when the corresponding toggles are enabled.
-   [ ] Upload metadata (detail files) is correctly generated by all high-speed backends.

## Out of Scope
-   Changing the highlight color itself.
-   Adding path information to the web UI.
