# Specification: Refactor Mushaf Rendering Verification Script

## Overview
The goal of this track is to refactor and update the `scripts/verify_mushaf_rendering.py` script. This script serves as a verification tool and a testing ground for the Mushaf rendering engine. It must be updated to use the latest fonts, database structures, and Right-to-Left (RTL) layout logic to ensure it accurately reflects the core video generation functionality.

## Functional Requirements

### 1. Font Integration
- Update the script to use the TrueType fonts located in the `QPC_V2_Font.ttf` directory.
- Implement specific font selection for different line types:
    - **Surah Headers:** Use `QCF_SurahHeader_COLOR-Regular.ttf` (with `QCF_SURA.TTF` as a fallback).
    - **Bismillah:** Use `QCF_BSML.TTF`.
    - **Standard Lines:** Use the page-specific fonts (`p1.ttf`, `p2.ttf`, etc.).

### 2. Layout and Directionality
- Ensure that Arabic words are sequenced from Right-to-Left within each line, matching the behavior of the core rendering engine.
- Implement robust word-to-line mapping using the `qpc-v2-15-lines.db` (for layout) and `word_by_word_qpc-v2.db` (for word content) databases.

### 3. Data Sourcing
- Use `@schema_report.md` as a general reference for database table structures and column names.
- Correctly retrieve `surah_name`, `basmallah`, and `ayah` line types from the layout database.

### 4. Verification Logic Refinement
- Fix existing bugs in the rendering and verification logic within `scripts/verify_mushaf_rendering.py`.
- Ensure the script can be used to reliably test refinements to the rendering engine before they are integrated into the main application.

## Non-Functional Requirements
- The script's output should be visually consistent with the videos generated by the main application.
- Performance: The script should render verification frames or clips efficiently.

## Acceptance Criteria
- Running `python scripts/verify_mushaf_rendering.py` generates images or clips where:
    - Arabic word order is correct (RTL).
    - Surah headers and Bismillah use the correct decorative fonts.
    - Page fonts match the current Mushaf page being rendered.
- The script executes without database errors or font loading failures.

## Out of Scope
- Integration of this script into the web UI or the main job worker (it remains a standalone verification tool).
