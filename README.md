# PyGen - Python-based Islamic Video generating application

The application, named PyGen, is a Python-based web application built with FastAPI. Its primary purpose is to automate the generation of Islamic videos.

### Key Features:
-   **Video Generation:** Combines Quranic recitations, translations, and customizable visual elements to create engaging video content.
-   **Web Interface:** Provides a user-friendly interface for managing various aspects of video creation.
-   **Job Management:** Allows for queuing and managing video generation tasks as jobs.
-   **YouTube Integration:** Includes features for integrating with YouTube, likely for uploading generated videos and managing playlists.
-   **Configurable:** Uses a database-backed configuration system for various settings.
